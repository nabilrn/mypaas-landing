# MyPaas Public

Standalone Astro website for the public MyPaas landing page, documentation, and privacy policy.

- `/` — product landing page
- `/docs/` — documentation overview
- `/docs/installation/` — VM installation and Cloudflare setup
- `/docs/deployment-models/` — Dockerfile, Compose, and static contracts
- `/docs/configuration/` — identity, environment, webhook, and routing configuration
- `/docs/operations/` — status, logs, metrics, lifecycle, and rollback
- `/docs/security/` — trust model and security controls
- `/docs/limitations/` — explicit product boundaries
- `/privacy/` — public website and self-hosted instance privacy policy
